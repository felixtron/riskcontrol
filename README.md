# Siento44 Construcciones - Sitio Web

Este es el sitio web oficial de Siento44 Construcciones, desarrollado con Next.js, Tailwind CSS y ShadCN UI.

## Cómo subir este proyecto a tu repositorio de GitHub

Si ves el mensaje `nothing to commit, working tree clean`, asegúrate de seguir estos pasos exactamente:

1. **Abre una terminal** en la carpeta del proyecto.
2. **Inicializa el repositorio** (solo si no lo has hecho):
   ```bash
   git init
   ```
3. **Agrega el repositorio remoto** (reemplaza con tu URL si es diferente):
   ```bash
   git remote add origin https://github.com/felixtron/riskcontrol.git
   ```
4. **IMPORTANTE: Prepara los archivos**:
   ```bash
   git add .
   ```
5. **Realiza el commit**:
   ```bash
   git commit -m "Initial commit: Proyecto Siento44 Construcciones"
   ```
6. **Sube el código**:
   ```bash
   git branch -M main
   git push -u origin main
   ```

## Desarrollo Local

Para ejecutar el proyecto localmente:

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador para ver el resultado.

# Siento44 Construcciones - Sitio Web

Este es el sitio web oficial de Siento44 Construcciones, desarrollado con Next.js, Tailwind CSS y ShadCN UI.

## Cómo subir este proyecto a tu repositorio de GitHub

Para conectar este código con tu repositorio en `https://github.com/felixtron/riskcontrol.git`, sigue estos pasos en tu terminal local:

1. **Descarga el código** de este proyecto a tu computadora.
2. **Abre una terminal** en la carpeta del proyecto.
3. **Inicializa el repositorio local** (si aún no lo has hecho):
   ```bash
   git init
   ```
4. **Agrega el repositorio remoto**:
   ```bash
   git remote add origin https://github.com/felixtron/riskcontrol.git
   ```
5. **Agrega todos los archivos**:
   ```bash
   git add .
   ```
6. **Realiza tu primer commit**:
   ```bash
   git commit -m "Initial commit: Proyecto Siento44 Construcciones"
   ```
7. **Sube el código**:
   ```bash
   git branch -M main
   git push -u origin main
   ```

## Desarrollo

Para ejecutar el proyecto localmente:

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador para ver el resultado.
